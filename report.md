# 1. Introduction
Managing daily expenses is important for financial planning. Many people spend money without tracking their spending habits. The Personal Expense Tracker is a Python-based application that helps users record and manage their expenses efficiently.

This system allows users to add expenses, categorize them, and view their total spending. The expense data is stored in a CSV file so that it can be accessed later.

---

# 2. Objective of the Project

The main objectives of this project are:

- To develop a simple expense tracking system using Python.
- To record daily expenses.
- To store expense data in a CSV file.
- To calculate total expenses automatically.
- To visualize expenses using graphs.

# 6. Working of the System

## Step 1: Add Expense
The user enters:
- Expense name
- Category
- Amount

The program stores this information in the CSV file.

## Step 2: View Expenses
The system reads the CSV file and displays all recorded expenses.

## Step 3: Calculate Total Expense
The program calculates the total expense by summing all values.

## Step 4: Graph Visualization
The system generates a bar graph showing expenses by category using Matplotlib.
